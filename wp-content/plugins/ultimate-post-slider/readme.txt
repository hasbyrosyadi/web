A Post Slider Widget based on bxslider.

== Description ==

A Post Slider Widget built around what you need to do, easlily the best Wordpress Post Slider Widget there is to use to display your posts, sticky posts or even both, With lots of features, this widget is easily manipulated and gives the user a great experience with any kind of post, with the ability to feature posts in a news like fashion based on bxslider.

This widget was designed to be easy to use for both the average user and the developer type.

= Options =

* Show Titles
* Link Titles
* Show post Excerpt
* Set excerpt Length
* Show Category
* Show Readmore
* Custom Readmore Title
* Set number of posts to show
* Show Thumbnails
* Display Sticky posts, all posts or hide sticky posts
* Display posts from categories
* Ability to multi-select categories
* OrderBy - Date, Title, Comment Count or random
* Add custom css classes and ID's
* Enable slider
* With options like - Speed, random, adaptive height and loads more.
** Display Custom Fields.
